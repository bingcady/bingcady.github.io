Optimizing Web Pages

View each of the two following pages, and note the delay in displaying each page.

  • Non-Optimized Page

  • Semi-Optimized Page

These pages are long and complex, so even the Semi-Optimized Page will load somewhat slowly. But whereas the Non-Optimized Page takes 41 seconds to load (over a 56k modem line) and a bit longer to display, the Semi-Optimized page takes 29 seconds. Better, but still way slow.

The following page takes advantage of the changes made to the Semi-Optimized page, and then chunks the big software list into a number of pages which each contain part of the list.

  • Optimized Page

This page loads in just 5 seconds over a 56k line. Much better! Every second of waiting on the Web means the loss of some of your visitors, so optimizing your pages can greatly improve user satisfaction.
