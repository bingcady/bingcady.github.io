Navigation

Digital Dimensions

http://www.dimensions.com/

Pointlessly redundant navigation: all icons go to the same page. Then, a break of the expectations they’ve set in the mind of the user: on the next page, the same set of icons again goes to a single page, but it’s now different.

 

Joseph Campbell Foundation site map

http://www.jcf.org/meta_end.htm

There is no at-a-glance navigation: you have to mouseover the it feels like a version of the game concentration. This is what Vincent Flanders calls Mystery Meat Navigation.

This page also loads VERY slowly, due to too many graphics (doubled by the need to also download the mouseover versions of the graphics)

 

Music Online

http://www.musicon-line.com/content.html

In order to see what the left menus mean, you have to mouseover and look in the status bar.
